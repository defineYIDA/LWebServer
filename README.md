# 通过搭建简单HTTP服务器，学习服务器原理

## 学习内容：
- IO模式
- Socket
- TCP/IP
- 框架源码(Tomcat,Spring,Netty等)

## 实现的简化功能：

- HTTP Protocol
- Servlet
1) ServletContext
2) Dispatcher
- Request&Response
- Forward&Redirect
- session&cookie

## 使用技术

基于Java BIO、多线程、Socket网络编程、XML解析、log4j/slf4j日志

## 准备添加的功能：
- NIO实现多路复用
- Filter
- Listener
- SSL
- limtlatch
- APR
## 模式
- 当前NIO的实现参照的是Tomcat的源码
